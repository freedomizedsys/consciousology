# Consciousness Studies v7  
**Existence Operating System — A Dynamic Unified Framework for Consciousness Generation, Collective Coexistence, and Cosmic Evolution**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

This repository contains the numerical simulation code, Jupyter Notebooks, and figures for the paper **"Consciousness Studies v7"**.

## Paper Abstract

Consciousness Studies v7 proposes a unified dynamic framework that formalizes consciousness as the universe's optimal evolutionary protocol for self-observation, asymmetry maintenance, and complexity generation. The framework consists of four core equations, extended with critical thresholds, optimal control paths, mean-field collective dynamics, and a holographic multi-scale architecture.

The quantum "can/cannot" state is precisely mapped to macroscopic consciousness intensity via expectation values, establishing a seamless continuity from quantum micro-scale to cosmic macro-scale. The model applies to both closed and open universes. Numerical simulations demonstrate that explosive phase transitions occur independently of boundary conditions, resolving the observer paradox and the problem of infinite dilution.

This framework simultaneously addresses four fundamental questions — how to exist, how to coexist, how to evolve, and the meaning of existence — transforming consciousness research from speculative philosophy into a computable, optimizable, and participatory science.

## Repository Contents

- `consciousness_v7_simulation.py` — Main simulation script (RK4 adaptive integrator, Monte Carlo scanning, Bayesian fitting)
- `figures/` — Four figures used in the paper
- `notebooks/` — Jupyter Notebooks (to be added)
- `requirements.txt` — Python dependencies

## How to Reproduce the Simulations and Figures

```bash
# 1. Clone the repository

# 2. Install dependencies
